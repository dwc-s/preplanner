Add overlays from a WMS server

Paste a WMS endpoint and pick the layers you want — no GIS knowledge needed. For Massachusetts, try https://gis-prod.digital.mass.gov/geoserver/wms.

Each layer you add becomes its own toggleable overlay on the map (transparent, so it won’t hide the basemap).

Import GIS file

GeoJSON, KML, GPX, a zipped Shapefile, or the loose Shapefile parts — select the .shp, .dbf and .shx together, plus .prj so it can be placed correctly (projected shapefiles are reprojected to lat/lon automatically). Points → access points, lines → routes, polygons → custom. Up to 2000 features per import.
